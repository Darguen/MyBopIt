Aplicación móvil que adapta al famoso juego de Bop It, en este caso la aplicación para usar
las características de movimiento o pasos a seguir va a usar una serie de funcionalidades
del teléfono, tales como giroscopio, tocar la pantalla, entre otros.

Este comprimido contiene el APK, el enlace del repositorio en github, y el link a la grabación
asociada.
